Para a criação do smartforms vocÊ precisará criar uma estrutura com campos fixos, para campos dinâmicos dentro dessa estrutura deverá criar uma categoria de tabela.

      Campos Fixos = Campos que se repetem em todas as telas Ex: cabeçalho e rodapé.

      Campos dinâmicos = Campos que podem vir mais de uma informação ex: itens.
