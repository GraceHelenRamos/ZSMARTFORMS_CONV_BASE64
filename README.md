Para Criação do smartforms, será necessário criar uma estrutura pensando em quais campos serão fixos e quais serão dinâmicos.

    Campos Fixos = Campos que se repete em todas as páginas exemplo: cabeçalho e rodapé.
  
    Campos dinâmiscos =  Retornos que podem vir mais de uma informações exemplos: itens.
